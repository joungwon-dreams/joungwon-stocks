# PDF Report Structure (v1)

This document defines the standard layout and structure for the stock analysis PDF report generated by `scripts/gemini/generate_pdf_report.py`.

**Latest Update:** 2025-11-27
**Style:** Generative AI Equity Research (Sample.pdf based)

## Page Structure

### Page 1
1.  **Header:** Stock Name, Code, Date, Time, Page Number
2.  **Sub-header:** "Generative AI Equity Research"
3.  **Investment Opinion Table:**
    *   Columns: Investment Opinion (Emoji + Text), Current Price (No KRW), Average Price (Profit% Color), Target Price (No KRW), Upside (Arrow + Color)
4.  **Key Metrics:** PER, PBR, ROE, Dividend Yield, Market Cap, Foreign Ratio, Sector, 52W High
5.  **Company Overview:** Summary text

### Page 2
6.  **Investment Consensus:**
    *   Header: "투자의견 컨센서스 (기준:YY.MM.DD)"
    *   Content: Gauge Chart (Left), Detail Table (Right)
    *   Source: www.wisereport.co.kr
7.  **Analyst Targets:**
    *   Header: "증권사 목표가"
    *   Table: Date, Target Price, Change (Pre.), Opinion (Mini Bar Chart), Broker, Report Title
8.  **Recent 2-Week Trend:**
    *   Chart: Price trend (Last 14 days)
    *   Features: Avg Buy Price Line (Purple), Y-axis Commas, Height 5.5cm

### Page 3
9.  **Holding Status:**
    *   **PageBreak Before:** Yes
    *   Table: Current holding details
10. **Real-time Ticks:**
    *   Header: "Real-time Ticks"
    *   Dashboard: Top Summary (Open, Curr, Vol, High, Low, Avg, Eval, Profit) + Detail List (34 items, Time <= 15:30)

### Page 4
11. **Price Trend (Long-term):**
    *   Header: "Price Trend (주가 추이)"
    *   Chart: 120-day OHLCV + Avg Buy Price Line

### Page 5
12. **Financial Performance:**
    *   Header: "Financial Performance (재무 실적)"
    *   Chart: Revenue/Op.Profit/Net Profit (Bar Chart), Height Reduced (2.8cm)
    *   Table: Yearly Financials
13. **Investor Trends:**
    *   Header: "Investor Trends (수급 동향)"
    *   Chart 1: 30-day Net Buying (Subplot: Top=Price, Bottom=Vol)
    *   Chart 2: 1-Year Cumulative Net Buying (Subplot: Top=Price, Bottom=Cum Vol)

### Page 6
14. **Peer Comparison:**
    *   **PageBreak Before:** Yes (to prevent splitting)
    *   Header: "동종업계 비교"
    *   Chart: Peer Comparison Bar Chart
    *   Table: Peer Metrics

### Page 7+
15. **News:**
    *   Header: "최근 뉴스 분석"
    *   Content: News Cards (Title, Date, Source, Sentiment, Summary)
16. **Footer:**
    *   Text: "본 리포트는 AI 기반 자동 생성 리포트입니다..."

## Maintenance Guide

*   **Script:** `scripts/gemini/generate_pdf_report.py`
*   **Components:** `scripts/gemini/components/`
*   **Layout Changes:** Always update this document when changing section order or adding new sections.
