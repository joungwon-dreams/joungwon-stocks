# 작업 규칙 (Workflow Rules)

## 문서 확인 요청 시 절차

사용자가 "확인해", "v1 확인해" 등 문서 확인을 요청하면:

1. **문서 내용 확인** - 해당 문서를 읽고 내용 파악
2. **내용 요약 및 의견 제시** - 새로운 지시사항 요약, 구현 방향에 대한 의견 제시
3. **작업 시작 여부 확인** - "진행할까요?" 등으로 사용자에게 확인
4. **승인 후 작업 진행** - 사용자가 "진행해", "시작해" 등 승인하면 그때 구현 시작

### 금지 사항
- 문서 확인 요청만으로 바로 구현 시작하지 않음
- 사용자 승인 없이 코드 작성하지 않음

### 예시 흐름
```
User: "v1 확인해"
Claude: [문서 읽기] → "새로운 지시: Phase 4.5 글로벌 매크로... 
        의견: yfinance 활용이 적절해 보입니다. 진행할까요?"
User: "진행해"
Claude: [구현 시작]
```
